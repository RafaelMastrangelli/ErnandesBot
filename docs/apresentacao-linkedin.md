Desenvolvendo um sistema de documentação automática para repositórios Git com IA! Conheça o ErnandesBot, uma aplicação Node.js que utiliza inteligência artificial para gerar documentação automática com base no código-fonte. Com uma arquitetura modular, ele é composto por três serviços principais: API, fila de processamento e worker. O ErnandesBot utiliza os serviços de IA da OpenAI ou Groq e suporta provedores como GitHub e Bitbucket.

Com o ErnandesBot, você pode:

* Gerar documentação automática para seus repositórios Git
* Configurar a aplicação por meio de variáveis de ambiente
* Utilizar a fila de processamento do RabbitMQ para armazenar eventos de processamento
* Commitar a documentação gerada no repositório Git

A tecnologia utilizada inclui Node.js, RabbitMQ, OpenAI e Groq. Com o ErnandesBot, você pode aumentar a produtividade e melhorar a qualidade da documentação dos seus projetos.
